# Fallback Topic Pool

> **What this file is for:** when the daily run finds no pending row in your sheet, it picks one topic from this file and writes a post about it. This is your "I have nothing prepared today, but I still want to post" safety net.
>
> **Format:** one topic per line. Lines starting with `#` (headings) and blank lines are ignored. The script avoids picking the same topic twice within the most recent 10 fallbacks, so a list of 15-30 topics gives healthy variety.
>
> **What makes a good fallback topic:** a specific lens or claim, not a category. "AI" is a category; "the gap between AI demos and AI in production" is a topic. The post-generator will turn each line into a 100-150 word post, so each line should be specific enough that you'd recognize the resulting post as one *you* would write.
